XHTML （ The eXtensible Hypertext Markup Language ）翻譯為『可擴展超文本置標語言』 1.0。 XHTML1.0 是將 HTML 4修改為符合 XML 1.0語法的規格。 所以我們可以簡單地說，XHTML 是以 XML 技術 為基礎的 HTML，其使得 Web 世界朝向模組化（ Modular ）及可擴 展化，邁出了第一步。

++ 主要的區別
# XHTML文件的開始要聲明DTD。
# 你需要為你的<html>標籤添加一個命名空間。
# 所有的標籤和標籤的屬性都必須小寫，屬性值可以大寫。
# 屬性值必須用引號括起來。單引號雙引號均可。
# 所有的標籤都必須被關閉，空標籤也不例外。
# XHTML文件必須有一個根元素(根標籤)，而且必須是html
# XHTML元素(標籤)必須正確嵌套。
