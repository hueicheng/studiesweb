簡介

HTML的全名是HyperText Markup Language，照字面上翻譯為『超文件標記語言』，是編寫網頁的基本語言，而它並不是一個程式， 只是一些插在普通文件內的碼( code )，這些碼可以控制我們的瀏覽器要怎樣把文件顯示出來。

一個HTML檔稱為HTML document，存檔的副檔名為htm或html，使用簡單的文書編輯軟體（例如 pe2、vi、記事本、小作家或 Word）就可以編寫。

所謂「超文件」(Hypertext) 藉由一些特殊標籤（例如連結）的協助，可以讓你以樹狀甚至網狀來組織你的文件。

標記 (Markup) 是透過在文章中插入標籤 (Tags) 來賦予文字一些特性，如標題，段落，連結等等。
