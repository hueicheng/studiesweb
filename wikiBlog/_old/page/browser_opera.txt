Opera起初是一款挪威 Opera software ASA 製作的葉面邊謙是瀏覽器，由於新版本的Opera增加了大量的網路功能，官方將 Opera 定義為一個網路套件。

Opera 支援相當多的作業系統，像windows、MAC OS X、linux、freeBSD 。

另外 Opera 還有手機版本 Opera mini、Opera mobile，以及提供NDS和WII的瀏覽器 Opera devcies。

2008年底 Opera 推出了中國版本命名為「朱雀」，並在台灣成立分公司也投入了不少資源再台灣市場也加入了微網誌噗浪，展現其對於正體世界的市場的企圖。

2009年，由台灣網友推出了 Opera 台灣版，名為「三太子」（Nalakuvara），並於2010年獲得官方認證。

Opera 這款瀏覽器擁有許多不同的功能，剛打開 Opera 時我注意到他的預設介面跟 Google一樣乾淨，只是多了左方跟下下方的介面，在左方的介面中 Opera 把書籤跟下載管理、瀏覽紀錄丟到了這邊。

這三樣算是一般使用者常用的功能，當然還不只這些除了上述三樣還有筆記、widget(套件取得)、unite(網路主機？)，另外這三項一般使用者可能不常用的，不過有興趣去玩的話會發現這三樣功能有也是滿好玩的功能。筆記這項功能就跟名稱一樣，基本上你要是在瀏覽忠看到有趣的東西想記錄下來又懶得開記事本的話 Opera 貼心的幫你把記事本丟到瀏覽器裡面了，這筆記可以儲存還可以寄出去分享給你的朋友。

widget 這功能可以幫助使用者取得套件，當然，套件總類也是琳瑯滿目包括了Opera 提供了跟網路上使用者開發的，不過我個人倒是沒怎麼使用，畢竟每個人的需求都不同。

unite 這功能由 Opera 介紹可以使自己的瀏覽器變成網路主機，讓自己成為伺服器同時自己也是客戶端，可以選擇與誰分享也可以分享他人的東西只要兩人都有開起這個功能；點開按鈕後裡面還有其他功能，不過至些功能可都是要擁有一個 Opera 的帳號才能使用的；加入後可以和朋友們分享檔案、照片、音樂、網頁、留言，不過要注意的是一旦關閉 Opera 你的應用程式就會無法執行，這相當於將伺服器端關機無論客戶端怎麼發出回應你都收不到了。

介紹完左邊介紹下上方跟下方的應用，Opera 將選單跟分頁列做在最上方，這點跟Google很像
先用過 Google 再用 Opera 的人就會有這樣的想法，不過聽說這樣的做法 Opera 可說是第一人
Google 只是借來用用，連火狐4 也都借去玩了…
選單裡面就是一些常用到的東西，設定、書籤一些哩哩摳摳的東西都在這，其中還有一項同步功能，沒錯！跟Google一樣地同步功能，也是同步差不多的東西用過Google的就知道。
在分頁列最右邊有個回收桶，在 Opera 中常常可以看到，大概是 Opera 認為使用者可能有一堆東西要丟，丟完了可能又想撿取回來吧..
既然是放在分頁列，裡面裝的當然是已經關閉的分頁被擋住的廣告視窗好像也會丟到這裡。

再來是下方 Opera 下方也放了點東西，不過下方的東西都是快捷按鈕，包刮了打開面板、伺服器端、同步跟加速，這加速倒是讓我很感興趣的東西；一開始以為是個會讓網速提升的東西
但事實證明我錯了，所謂加速大概是在眾多網路功能中先將其餘的網路功能流量下降然後加速 Opera 的瀏覽速度，不過這只是我的解釋
Opera 說明中提到，它的加速是讓數據傳輸加速、減少數據量，通過圖像壓縮下採樣不過這麼做可能會使得圖像失真。
我自己倒是不喜歡開這項功能，不知為何開了之後反而讓我覺得瀏覽速度變慢？！

**語音**

介紹一下其他功能其中最吸引我注意的是 Opera 的語音功能
因為這項功能在其他的瀏覽器上並沒有看到，這說明 Opera 對於肢體殘障人士比較有愛心？！
可以使用語音的功能來操控 Opera，畢竟隨著科技的進步人類也越來越懶惰能一邊吃著零食一邊用嘴巴控制瀏覽器觀看想看的網頁不是一件很棒的事情嗎？
不過Opera 這項功能做的還不是很完善，他需要標準的語音來控制 Opera 還只能用英文(所以一邊吃零食是不可能的事情..)至於中文這方面，大概還沒有替中文使用者的設想所以英文是很重要低..
Opera 也預設了對應的語音，可以選擇男女聲在使用者指令不當時提出錯誤訊息，不過聲音很平板就是。
目前 Opera 提供的語音功能不多，而且開頭一定要加 Opera 不然他不想理你
舉幾個例子：Opera speak 這個裡另可以唸出選取中的文字，Opera home 就可以回到你的首頁
Opera back/forward 則可以上下頁切換。

這功能雖然很方便，但是之前說過了這功能並沒有很完善，像我之前想要叫 Opera 幫我搜尋
於是我說：「Opera use Google search Kimt」就沒有理我…
可能這是未來科技才有辦法達到的水準，現在這麼做太勉強他了…
不過我自己有測試一些可行但說明上沒有寫的語音指令，如：Opera Google、Opera Facebook
這兩個指令不知道是不是因為我常瀏覽或是已有紀錄所以可以啟動。

**快速撥號**

這功能有點特別並且可以搭配語音指令使用，快速撥號是 Opera 的預設首頁
也就是一開起就會看到快速撥號的頁面，一開始甚麼都沒有大概因做做成九宮格數字鍵所以被稱作快速撥號，亦或是翻譯問題(？)，快速撥號搭配 ctrl +數字鍵可以開啟被設定在快速撥號上的網頁，例：在快速撥號1中設定 yahoo 首頁只要按下 ctrl + 1 即可開啟 Yahoo 。
也可以使用語音 Opera one 一樣可以開啟，這功能就跟 Google 新分頁中把常用網頁記錄下來一樣，只是多了鍵盤跟語音啟動。

**右鍵**

右鍵 Opera 的右鍵裡面有一些其他瀏覽器所沒有的功能，這些功能有時候讓我覺得滿實用的
像是定時重新整理，有些時候玩某些 web game 會用到為了要刷新概況查看是否有新資訊等，其實除了玩 web game 之外還有其他地方也可以用到，例如想要掌握論壇新文章資訊的時候可以用到。定時重新仔入這時間可以自己設定也可以用預設的定值來跑，其他廠牌了瀏覽器就沒有這項功能，大部分瀏覽器都需要使用者自行重新整理或是網站有新回應時才會自動重新載入(？)，其實會用到這項功能的人不多但 Opera 還是貼心的照顧到少部分的使用者。

另外就是以其他程式開啟這項，這應該算是比較特別的，在畫面上點選這項功能的時候可以用不同的瀏覽器來開啟這個頁面，會這麼做通常只有 IE 限定的網頁大概才會去使用到，畢竟 Opera 並沒有IE Tab 這項好用的套件，而是使用了其他程式來抓住 IE 引擎，但是這麼做的效果並不是很好，所以有這項功能加減補強了限電 IE 瀏覽的網頁功能吧

**載入**

Opera 再載入方面也有自己的一套，Opera 在載入時會先將所有元件依序的載入測試其可用性，這樣做雖然會造成讀取速度的降低也會讓一些比較心急的使用者不耐煩。
就連在分頁上面也是如此，假設一次用三個分業分別打開不同的 Flash ，Opera 只會先執行當前頁面的flash 當使用者切換到其他頁面的時候才會執行另一頁面的 Flash 。這點相對於其他瀏覽器這做法很不同，像是 Google 的話就是一次性載入所有程式、元件，這麼做雖然會讓畫面顯示得快些，但其缺點是一次大量性的載入資源常常會使的頁面當機、LAG 等情況發生。

**HTML 5**

Opera 在顯示 HTML 5 這方面擁有相當亮眼的成績，許多 HTML 5 的標籤在其他瀏覽器無法顯示的，在 Opera 都看的到。在我們所測試的程式碼當中唯有 Opera 可以執行及時運算的程式碼，其他的瀏覽器目前尚未支援這項程式碼。不過在 HTML 5 這方面 Firefox 也競爭的挺激烈的，相當多的程式碼在 FX 下也是可以執行的至於排名二三的則是 Google、Safari
最後則是IE，或許誰能先掌握住 HTML 5 的技術誰就能成為瀏覽器中的霸主吧？
